## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:

   ```bash
     npm install
   ```

   ```bash
    pip install -r requirements.txt
   ```

## Usage

1. Start the development server:

   ```bash
     npm run start
   ```

   or

   ```bash
     python .\src\main.py
   ```
